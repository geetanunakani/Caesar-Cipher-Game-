# Caesar Cipher Game 🔐

A simple Python program that implements the Caesar Cipher encryption and decryption technique.

## Features

* Encrypt messages using a shift key
* Decrypt encrypted messages using the same shift key
* Handles spaces and special characters
* User-friendly menu-driven interface

## How It Works

The Caesar Cipher is a substitution cipher where each letter in the plaintext is shifted by a fixed number of positions in the alphabet.

### Example

**Input:**

* Message: `hello`
* Shift Key: `3`

**Encrypted Output:**

* `khoor`

**Decrypted Output:**

* `hello`
